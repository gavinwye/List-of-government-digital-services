Data sources

Contributing

This is an open, crowdsourced effort. If your service isn't on the list, you can add it.

We'll keep a record of who added the service but won't publish this information.

You'll need the:

  • service name
  • service URL
  • department that's responsible for the service

Only services with a 'pingable' URL will be included in the list.

By email

Email listofservices@digital.cabinet-office.gov.uk to add your service information to the list.

By pull request on Github

Open a pull request on the List of services repository.
