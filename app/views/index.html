Once we know where all of the servcies are we can work togethet in a more joined up way.

Contributing

This is an open croud-sorced effort if your service isn't on the list you can contribute it.

You need four peices of information:

  • The service name
  • The URL of the service
  • The department that's respobnsible for the service

Only services with a pingable URL will be included in the list.

By email

Email registerofservices@digital.cabinet-office.gov.uk with the details above and they will be included in the list.

By pull request on Github

Open a pull request on the Register of services reposiotory.
